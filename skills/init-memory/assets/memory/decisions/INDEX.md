# Decisions Index

ADR-lite entries covering architectural choices that future agents need to
know about. Each entry's `Context at decision time` captures what was true
when decided -- read it before second-guessing.

Promotion rule: before writing a new decision, grep this directory for related
entries. If a supersession is needed, flip the old entry's `Status` and update
this INDEX in the **same commit** -- never write silently.
